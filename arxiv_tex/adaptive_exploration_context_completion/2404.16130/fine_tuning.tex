\subsection{Relationship fine-tuning procedure}

We used GPT-4o to synthetize synthetic data for fine-tuning.
Our approach used the following procedure:
\begin{enumerate}
    \item Given randomly sampled news topics, then prompted GPT-4o to generate a mock news story based on those topics.
    \item We prompted the LLM to extract entities and relationships from the article using the same prompt as we use for GraphRAG.
    \item Given the extracted entities and relationships, we then prompt the LLM to generate a new news story using those extracted entities and relationships. 
\end{enumerate}
The second generation of the mock news article helps ensure the all the entities and relationships are mentioned in the article.
The articles become the inputs to the fine-tuning, and the relationships become the labels.
Using this method, we created 29059 news articles and corresponding relationship labels.

All fine-tuning was done using Azure ML studio \cite{azure_ml_studio}.
