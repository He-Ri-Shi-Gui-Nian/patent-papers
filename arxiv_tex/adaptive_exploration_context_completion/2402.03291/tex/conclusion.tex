\section{Conclusion}
\label{sec:conclusion}
We contribute \system, an interactive widget suite
for composing  
that enables knowledge acquisition and integration tasks
within computational notebooks. 
% The widget suite facilitates the blending 
% of programming and interactive analysis ---
% an aspect that was highly appreciated by 
% data practitioners.
Compared to participants' previous experience,
\system widgets
enabled efficient executions of these tasks
by blending programming and interactive analysis.
Such a design provided a seamless experience to the 
users by eliminating tedious context switching.
Moreover, the multiple coordinated view-based
design of the widgets helped users 
uncover data quality issues
and effectively review agent recommendations. 
% \saj{However, similar to any new framework, \system
% exhibited several limitations which encourage 
% interesting future research.
In the short term, we continue to employ \system
to meet the needs of data practitioners in \company.
We plan to deploy \system to other workflows within
the HRKG platform within \company which would require
additional need finding and participatory design of widgets
and enhancement of the graph service library.
% Finally, we plan to address 
% the fundamental research questions related to balancing human agency and automation with respect to recommendation and composability, 
% conduct larger-scale studies to get feedback, 
% and expand to other domains for generalizability.
