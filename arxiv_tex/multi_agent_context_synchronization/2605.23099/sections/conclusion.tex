\section{Conclusion}

% MAD suffers from rapid context growth, which limits its scalability in larger multi-agent settings. Existing methods reduce communication by using prior signals about agent reliability, but these signals can become misaligned with response quality under hallucination.
We present \sysname, a Bayesian-inspired MAD framework that leverages posterior evidence from debate outcomes. \sysname first probes selected communications to estimate agent reliability, then incrementally constructs the communication graph toward agents with stronger posterior signals. Evaluation across multiple LLMs and datasets shows \sysname improves the cost-accuracy trade-off, reducing communication links and tokens by up to 75\% and 61\%, while matching or improving accuracy relative to the most accurate competing MAD baseline.
