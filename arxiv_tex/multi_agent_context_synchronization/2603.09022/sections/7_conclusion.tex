We addressed run-to-run variance in multi-turn, multi-agent LLM evaluation caused by compounding deviations and prompt sensitivity. We introduced \NAME{}, a weight-free self-play framework that couples \emph{retention}, a persistent memory bank distilling trajectories into reusable insights, with \emph{exploration}, tournament-style prompt evolution and prioritized replay. Across five text-based games, \NAME{} substantially improves win rates while using 19$\times$ fewer games than RL baselines, and reduces outcome dispersion. Ablation studies confirm both components are necessary. The learned contexts transfer across games and some model families. These findings suggest that substantial headroom in multi-agent LLM games can be unlocked through context optimization rather than weight updates.
