\section{Conclusion}

To enable more effective inference under IKG scenarios, we proposed DoM, a MAD framework for IKGQA. By coordinating structured and unstructured evidence through agent collaboration, DoM effectively leverages their complementary strengths. Additionally, we constructed the IKGWQ dataset by revisiting samples from CWQ and WebQSP. We rebuild the corresponding QA pairs using up-to-date knowledge retrieved from reliable sources. This provides a more realistic benchmark for evaluating IKGQA systems. Extensive experiments demonstrate that DoM consistently outperforms strong baselines across multiple settings. We released both the IKGWQ dataset and our code to support future research.\footnote[3]{\url{https://github.com/liujilong0116/DoM}}
